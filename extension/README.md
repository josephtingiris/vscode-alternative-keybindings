# AlterNative Keybindings

Intuitively use alt+arrow and alt+hjkl keyoard navigation across VS Code workbench editor groups, side bars, panel, and
terminal elements.

## Highlights

- Directional navigation featuring right and left handed layouts.
- Easy and fast Editor Group, Panel, and Sub-Panel (Terminal) pane cycling.
- Lightweight status bar indicator and optional edge notifications.

## Configuration

- `altKey.showStatusBar`: show the status bar indicator (default `true`).
- `altKey.logLevel`: `off`, `error`, `info`, `debug` (default `off`).
- `altKey.logChannel`: output channel name (default `AlterNative Keybindings`).
- `altKey.logFile`: optional file path for log output (default empty).

For more information, see the repository root README.md.
